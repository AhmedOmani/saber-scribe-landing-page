# The Saber Scribe - Landing Page

This is the official landing page website for **The Saber Scribe**, an AI-powered desktop application for English teachers.

**Live Website:** [https://saber-scribe-landing-page.vercel.app/](https://saber-scribe-landing-page.vercel.app/)

## Main Project
The source code for the actual desktop application can be found here:

This is a [Next.js](https://nextjs.org) project built with Tailwind CSS and Shadcn UI.

```bash
npm run dev
```
